# Нормативы ЦБ России для банков — полный конспект

## Что такое пруденциальные нормативы

Пруденциальные нормативы — обязательные количественные ограничения, устанавливаемые ЦБ РФ для банков. Их цель: не допустить принятия избыточных рисков, которые могут привести к банкротству банка и потерям вкладчиков.

**Правовая база:**
- Федеральный закон № 86-ФЗ «О Центральном банке Российской Федерации»
- Федеральный закон № 395-1 «О банках и банковской деятельности»
- **Инструкция ЦБ РФ № 199-И** — основной документ по обязательным нормативам банков
- Положения ЦБ по расчёту капитала (№ 646-П), кредитному риску, рыночному риску

---

## Карта нормативов ЦБ

```
Нормативы ЦБ РФ
│
├── Нормативы достаточности капитала
│   ├── Н1.0 — общая достаточность капитала
│   ├── Н1.1 — базовый капитал (CET1)
│   ├── Н1.2 — основной капитал (Tier 1)
│   └── Н1.4 — для банков с базовой лицензией
│
├── Нормативы ликвидности
│   ├── Н2 — мгновенная ликвидность
│   ├── Н3 — текущая ликвидность
│   └── Н4 — долгосрочная ликвидность
│
├── Нормативы концентрации (кредитный риск)
│   ├── Н6 — максимальный риск на одного заёмщика
│   ├── Н7 — максимальный размер крупных кредитных рисков
│   ├── Н9.1 — риск на акционеров
│   └── Н10.1 — риск на инсайдеров
│
├── Другие нормативы
│   ├── Н12 — использование капитала для приобретения акций
│   └── Н25 — риск на связанные стороны
│
└── Буферы капитала (Basel III)
    ├── Буфер консервации (2,5%)
    ├── Контрциклический буфер (0–2,5%)
    └── Буфер СЗКО (1–3,5%)
```

## Раздел 1. Нормативы достаточности капитала

### Логика: что такое достаточность капитала

```
Достаточность капитала = Капитал / Активы, взвешенные по риску (RWA)
```

**RWA (Risk-Weighted Assets)** — не просто сумма активов, а активы с учётом их рискованности:
- Кэш и ОФЗ → вес 0% (безрисковые)
- Кредиты корпорациям → 100%
- Розничная ипотека → 35–100% (в зависимости от LTV)
- Необеспеченные потребкредиты → 150%+

Чем выше риск актива — тем больше капитала требуется под него держать.

---

### Н1.0 — Норматив достаточности общего капитала

```
Н1.0 = Капитал (Tier 1 + Tier 2) / RWA ≥ 8%
```

**Капитал:**
- **Tier 1 (основной капитал)** = Базовый (CET1) + Добавочный (AT1)
- **Tier 2 (дополнительный капитал)** = субординированный долг, часть резервов

**Структура капитала банка:**
```
Tier 1 (основной)
├── CET1 (базовый) — акционерный капитал + нераспределённая прибыль − нематериальные активы
└── AT1 (добавочный) — бессрочные субординированные инструменты (AT1 bonds, привилегированные акции)

Tier 2 (дополнительный)
└── Субординированные кредиты/облигации (срок ≥ 5 лет) + часть резервов
```

---

### Н1.1 — Норматив достаточности базового капитала (CET1)

```
Н1.1 = CET1 / RWA ≥ 4,5%
```

**CET1 — самый «чистый» капитал:**
- Уставный капитал (обыкновенные акции)
- Эмиссионный доход
- Нераспределённая прибыль
- **Минус:** нематериальные активы, гудвилл, отложенные налоговые активы, вложения в другие банки

---

### Н1.2 — Норматив достаточности основного капитала (Tier 1)

```
Н1.2 = (CET1 + AT1) / RWA ≥ 6%
```

---

### Н1.4 — Для банков с базовой лицензией

```
Н1.4 ≥ 8%  (аналог Н1.0 для небольших банков с ограниченным кругом операций)
```

---

### Буферы капитала (сверх минимальных нормативов)

| Буфер | Размер | Назначение |
|-------|--------|-----------|
| **Буфер консервации** | +2,5% к CET1 | Постоянный. Нарушение → ограничение на дивиденды и бонусы |
| **Контрциклический буфер (CCyB)** | 0–2,5% | Вводится ЦБ в периоды перегрева кредитного рынка. Сейчас в РФ = 0% |
| **Буфер СЗКО** | 1–3,5% | Для системно значимых кредитных организаций (ВТБ, Сбер, Газпромбанк и др.) |

**Итого минимальный уровень CET1 для СЗКО:**
```
4,5% (Н1.1) + 2,5% (буфер консервации) + 1% (буфер СЗКО) = 8% CET1
```

---

### Связь с показателями банка

| Показатель | Связь с нормативами капитала |
|------------|------------------------------|
| **ROE** | Давление на капитал → банк вынужден балансировать между ростом и достаточностью |
| **Рост кредитного портфеля** | Увеличивает RWA → снижает Н1.0 → нужна прибыль или допэмиссия |
| **Дивиденды** | Выплата дивидендов снижает капитал → снижает Н1.0 |
| **Резервы под кредитные потери** | Досоздание резервов снижает прибыль → снижает капитал → снижает Н1.0 |
| **Рискованность активов** | Рост доли потребкредитов (вес 150%) быстрее «съедает» капитал, чем ипотека (вес 35%) |
| **Субординированный долг** | Привлечение субордов пополняет Tier 2 → улучшает Н1.0, но не Н1.1 |
| **AT1 инструменты** | Дополнительный уровень, улучшает Н1.2 без размытия акционеров |

## Раздел 2. Нормативы ликвидности

### Логика: что такое ликвидность банка

Ликвидность — способность банка своевременно и в полном объёме выполнять денежные обязательства. Банк может быть прибыльным, но неликвидным — если все клиенты одновременно потребуют деньги, а активы долгосрочные.

---

### Н2 — Норматив мгновенной ликвидности

```
Н2 = ВЛА_мгновенные / Обязательства_до_востребования ≥ 15%

ВЛА_мгновенные = высоколиквидные активы (кэш + корсчёт в ЦБ + банковские металлы)
Обязательства до востребования = депозиты до востребования + расчётные счета
```

**Смысл:** банк должен немедленно покрыть минимум 15% требований «до востребования» живыми деньгами.

**Связь с показателями:**
- Рост остатков на расчётных счетах корпораций → растёт знаменатель → давление на Н2
- Банк вынужден держать часть активов в кэше или на корсчёте ЦБ (неработающие активы) → снижает NIM
- Компромисс: чем выше Н2 — тем ниже доходность активов

---

### Н3 — Норматив текущей ликвидности

```
Н3 = Ликвидные активы (до 30 дней) / Обязательства (до 30 дней) ≥ 50%
```

**Ликвидные активы до 30 дней:**
- Кэш, корсчёт в ЦБ
- МБК выданные сроком до 30 дней
- Ценные бумаги из ломбардного списка ЦБ

**Смысл:** в горизонте 30 дней должно хватать минимум половины от обязательств.

**Связь с показателями:**
- Банк с длинными активами (ипотека) и короткими пассивами (депозиты до 30 дней) испытывает давление на Н3
- Решение: держать буфер ликвидных активов (снижает NIM), привлекать долгосрочное фондирование
- Операции РЕПО с ЦБ — быстрый способ пополнить ликвидность при давлении на Н3

---

### Н4 — Норматив долгосрочной ликвидности

```
Н4 = Долгосрочные требования (>1 года) / (Капитал + Долгосрочные обязательства >1 года) ≤ 120%
```

**Смысл:** долгосрочные активы не должны более чем в 1,2 раза превышать долгосрочное фондирование + капитал. Ограничивает трансформацию сроков (выдавать длинные кредиты за счёт коротких депозитов).

**Связь с показателями:**
- Ипотечный банк с портфелем 20 лет, фондируемый краткосрочными депозитами → давление на Н4
- Выпуск ипотечных облигаций (ИЦБ) → удлиняет пассивы → улучшает Н4
- Секьюритизация → снимает длинные активы с баланса → улучшает Н4

---

### LCR и NSFR — базельские нормативы ликвидности

Введены в РФ для СЗКО и крупных банков.

```
LCR (Liquidity Coverage Ratio) = HQLA / Чистый отток за 30 дней ≥ 100%
HQLA = высоколиквидные активы (Level 1: ОФЗ, кэш; Level 2: корпоративные облигации с дисконтом)

NSFR (Net Stable Funding Ratio) = Стабильное фондирование (ASF) / Требуемое финансирование (RSF) ≥ 100%
```

| LCR vs Н3 | |
|---|---|
| **Н3** | Российский норматив, проще по структуре |
| **LCR** | Basel III, детальнее: стресс-сценарий оттока за 30 дней с весами по типам обязательств |

## Раздел 3. Нормативы концентрации (кредитный риск)

### Н6 — Максимальный размер риска на одного заёмщика / группу связанных

```
Н6 = Кредитный риск на заёмщика (или группу) / Капитал банка ≤ 25%
```

**Смысл:** банк не может вложить более 25% своего капитала в одного заёмщика. Цель — не допустить ситуации, когда дефолт одного клиента «убивает» банк.

**В риск включается:**
- Ссуды и займы
- Гарантии и поручительства выданные
- Лимиты по кредитным линиям
- Вложения в облигации заёмщика

**Связь с показателями:**
- Для крупных корпоративных банков (ВТБ, Газпромбанк) — основное ограничение роста
- При капитале 3 трлн руб. → лимит на одного заёмщика = 750 млрд руб.
- Консорциальные кредиты (синдикации) — способ обойти Н6, разделив риск с другими банками

---

### Н7 — Максимальный размер крупных кредитных рисков

```
Н7 = Сумма крупных кредитных рисков / Капитал ≤ 800%
```

**Крупный кредитный риск** — риск на одного заёмщика > 5% капитала банка.

**Смысл:** суммарная концентрация всех крупных заёмщиков не может превышать 8 капиталов.

---

### Н9.1 — Риск на акционеров

```
Н9.1 = Совокупный риск на акционеров (доля ≥ 5%) / Капитал ≤ 50%
```

Защита от кредитования «под себя» — банк не должен превращаться в кошелёк своих владельцев.

---

### Н10.1 — Риск на инсайдеров

```
Н10.1 = Совокупный риск на инсайдеров (менеджмент, члены совета) / Капитал ≤ 3%
```

---

### Н25 — Риск на связанные стороны (аффилированные лица)

```
Н25 = Совокупный риск на связанные стороны / Капитал ≤ 20%
```

Более широкое ограничение, чем Н9.1 и Н10.1 — включает все аффилированные структуры.

---

### Н12 — Использование капитала для приобретения акций

```
Н12 = Инвестиции в акции / Капитал ≤ 25%
```

Ограничивает вложения банка в уставные капиталы других компаний — банк не должен быть холдингом-инвестором за счёт клиентских денег.

## Раздел 5. Связь нормативов с ключевыми банковскими KPI

### Как нормативы ограничивают рост банка

```
Банк хочет выдать кредитов на 100 млрд руб.
↓
RWA вырастут примерно на 100 млрд (если вес риска 100%)
↓
Н1.0 снизится: Капитал / (RWA_старые + 100 млрд)
↓
Если Н1.0 приближается к минимуму → нужно наращивать капитал
↓
Источники капитала:
  1. Нераспределённая прибыль (снизить дивиденды)
  2. Допэмиссия акций (размывает акционеров)
  3. Субординированный долг (улучшает Н1.0, но не Н1.1)
  4. Продать часть активов / секьюритизировать
```

### Таблица: норматив → показатель → рычаги управления

| Норматив | На что влияет | Рычаги для банка |
|----------|---------------|-----------------|
| **Н1.0 ↓** | Ограничивает рост кредитного портфеля | Прибыль, допэмиссия, субординированный долг, секьюритизация, снижение рисковых весов |
| **Н1.1 ↓** | Давление на дивиденды и бонусы (буфер) | Только органическая прибыль или допэмиссия обыкновенных акций |
| **Н2 ↓** | Снижение кэша / корсчёта | Привлечение МБК overnight, сокращение обязательств до востребования, РЕПО с ЦБ |
| **Н3 ↓** | Нехватка ликвидных активов на 30 дней | РЕПО, удлинение пассивов, продажа/залог ценных бумаг |
| **Н4 ↑** | Избыток длинных активов при коротких пассивах | Секьюритизация ипотеки, выпуск долгосрочных облигаций, привлечение субордов |
| **Н6 ↑** | Концентрация на одном заёмщике | Синдикация, продажа части требования, наращивание капитала |

## Раздел 6. Инструменты регуляторного управления банком

### Макропруденциальные надбавки

ЦБ вводит надбавки к рисковым весам в перегретых сегментах:

| Сегмент | Надбавка |
|---------|----------|
| Необеспеченные потребкредиты (ПДН > 80%) | +60–100% к весу риска |
| Ипотека с LTV > 85% | +25–50% |
| Автокредиты | +25% (введена в 2024) |
| Кредиты в иностранной валюте | повышенные веса |

**ПДН (Показатель долговой нагрузки)** = Ежемесячные платежи по долгам / Ежемесячный доход заёмщика.

Чем выше ПДН — тем дороже для банка выдавать такой кредит с точки зрения капитала.

### Как это влияет на банк
- Банк с портфелем высокорисковой розницы «расходует» капитал быстрее
- Стимул переходить в менее рисковые сегменты (ипотека, корпоративные кредиты с хорошим рейтингом)
- NIM может быть выше у рискованных сегментов, но давление на капитал тоже выше → ROIC не обязательно лучше

## Раздел 7. Резервы под кредитные потери (РВПС)

Это не норматив, но напрямую влияет на капитал и нормативы.

### Положение ЦБ № 590-П (кредитные организации)

Банк обязан формировать резервы под каждый кредит в зависимости от категории качества:

| Категория | Описание | Резерв |
|-----------|----------|--------|
| **I (Стандартные)** | Нет признаков риска | 0% |
| **II (Нестандартные)** | Умеренный кредитный риск | 1–20% |
| **III (Сомнительные)** | Значительный риск | 21–50% |
| **IV (Проблемные)** | Высокий риск | 51–100% |
| **V (Безнадёжные)** | Нет шансов на возврат | 100% |

### Связь резервов с нормативами
```
Рост резервов → Снижение прибыли → Снижение капитала → Снижение Н1.0

Пример:
Банк досоздал резервов на 50 млрд руб.
Прибыль до налога снизилась на 50 млрд
Налог (20%) = 10 млрд
Чистая прибыль снизилась на 40 млрд
Капитал снизился на 40 млрд
Если RWA = 2 трлн руб., то Н1.0 снизится на 40/2000 = 2 п.п.
```

### CoR (Cost of Risk) — стоимость риска
```
CoR = Отчисления в резервы за период / Средний кредитный портфель
```
- CoR 1% — нормальная среда
- CoR 2–3% — стресс
- CoR > 4% — кризис / проблемный портфель

## Раздел 8. Регуляторный капитал vs Экономический капитал

| | Регуляторный капитал | Экономический капитал |
|---|---|---|
| **Кто устанавливает** | ЦБ РФ / Basel | Сам банк внутри |
| **Метод** | Стандартизированный (рисковые веса по таблице) | Внутренние модели (IRB, VaR, стресс-тесты) |
| **Цель** | Защита вкладчиков / система | Оптимизация отдачи на капитал |
| **ВПОДК** | Обязательная самооценка достаточности | Внутренний процесс |
| **ICAAP** | Basel II/III — внутренняя оценка адекватности капитала | Отчётность перед ЦБ в рамках Pillar 2 |

## Раздел 9. Нарушение нормативов — последствия

| Норматив | Нарушение → последствие |
|----------|-------------------------|
| Н1.0 < 8% | Предписание ЦБ, запрет на рост, план восстановления |
| Н1.0 < 6% | ЦБ вправе ввести временную администрацию |
| Н1.0 < 2% | Обязательный отзыв лицензии |
| Н2, Н3 < минимума | Предписание, штраф, ограничение операций |
| Нарушение Н6 | Штраф, предписание об уменьшении позиции |
| Буфер консервации не выполнен | Ограничение дивидендов, бонусов, выкупа акций |

## Раздел 10. Быстрая шпаргалка по нормативам

```
Н1.0  = Капитал (Tier1+Tier2) / RWA             ≥ 8%
Н1.1  = CET1 / RWA                               ≥ 4,5%
Н1.2  = Tier1 / RWA                              ≥ 6%

Н2    = ВЛА_мгнов / Обяз_до_востреб             ≥ 15%
Н3    = Ликв_активы(30д) / Обяз(30д)             ≥ 50%
Н4    = Долгосрочн_треб(>1г) / (Кап+Долг_обяз)  ≤ 120%

Н6    = Риск_1заёмщик / Капитал                  ≤ 25%
```
