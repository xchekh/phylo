## 2. История

| Год | Событие |
|-----|---------|
| **1990** | Основан как Банк внешней торговли СССР — для обслуживания внешнеэкономической деятельности |
| **1998** | Кризис — банк устоял, получил государственную поддержку |
| **2002** | Переименован в ВТБ (Внешторгбанк) |
| **2005** | Приобретение Гута-банка → создание ВТБ 24 (розничный банк) |
| **2007** | IPO на Московской и Лондонской биржах. «Народное IPO» — 131 000 частных инвесторов |
| **2012** | Приобретение Банка Москвы (крупнейшая сделка M&A в российском банкинге) |
| **2014** | Санкции США и ЕС после присоединения Крыма — закрытие западных рынков капитала |
| **2016** | Поглощение Лето Банка → трансформация в «Почта Банк» (СП с Почтой России) |
| **2018** | Слияние ВТБ 24 с головным банком ВТБ — единый универсальный банк |
| **2022** | Масштабные санкции США, ЕС, Великобритании. Отключение от SWIFT. Уход с зарубежных рынков |
| **2023–2024** | Рекордная прибыль на фоне высоких ставок, продолжение цифровизации |

## 3. Группа ВТБ — структура

ВТБ — это не один банк, а финансовый конгломерат.

```
Группа ВТБ
├── Банк ВТБ (ПАО) — головной банк, корпоративный бизнес
├── ВТБ Капитал — инвестиционный банк (M&A, DCM, ECM, трейдинг)
├── Почта Банк — массовый розничный сегмент (СП с Почтой России, 50%+)
├── Мультибанк — МСБ-платформа
├── ВТБ Лизинг — финансовый лизинг
├── ВТБ Факторинг
├── ВТБ Страхование — страховые продукты
├── ВТБ Пенсионный фонд — НПФ
├── ВТБ Регистратор
└── ВТБ Инфраструктурный холдинг (проектное финансирование)
```

## 4. Бизнес-сегменты

### 4.1 Корпоративно-инвестиционный банк (КИБ)
- Кредитование крупного бизнеса и госкорпораций
- Проектное финансирование (инфраструктура, энергетика, ВПК)
- Торговое финансирование и аккредитивы
- Инвестиционный банкинг: организация размещений облигаций (DCM), акций (ECM)
- Управление ликвидностью корпораций (cash management)
- Деривативы, валютные операции, структурные продукты
- **Клиенты:** Роснефть, Газпром, Транснефть, РЖД, Ростех, Росатом

### 4.2 Средний и малый бизнес (МСБ)
- Кредиты на оборотный капитал и инвестиции
- Расчётно-кассовое обслуживание (РКО)
- Зарплатные проекты
- Эквайринг и платёжные решения
- Факторинг для МСБ
- Гарантии

### 4.3 Розничный бизнес
- **Ипотека** — один из крупнейших ипотечных банков России (2-е место после Сбера)
- Потребительские кредиты
- Автокредиты
- Дебетовые и кредитные карты
- Вклады и накопительные счета
- Брокерское обслуживание и инвестиции (ВТБ Мои Инвестиции)
- Страхование жизни (bancassurance)
- **Почта Банк** — охват малых городов через 38 000+ точек Почты России

### 4.4 Инвестиционный бизнес (ВТБ Капитал)
- M&A advisory
- Долговой рынок (организатор выпусков ОФЗ, корпоративных облигаций)
- Private Equity
- Управление активами (ВТБ Капитал Управление активами)

| ROE | ~20–25% |
| CIR (Cost-to-Income) | ~40–45% |
| NPL ratio | ~4–6% |

### Структура кредитного портфеля (ориентировочно)
```
Корпоративные кредиты      ~65–70%
Розничные кредиты          ~25–30%
  из них: ипотека          ~15–18%
  из них: потреб/авто/карты ~10–12%
МБК и прочее               ~3–5%
```

### P&L-лестница банка ВТБ (упрощённо)
```
Процентные доходы
− Процентные расходы
= NII (Чистый процентный доход)           ~60% всех доходов

+ Комиссионные доходы                     ~15–20%
+ Доходы от трейдинга и переоценки        ~10%
+ Прочие операционные доходы              ~5–10%
= Операционные доходы (Gross Revenue)

− Операционные расходы (персонал, IT, аренда)
= PPOP (Pre-Provision Operating Profit)

− Резервы под кредитные потери (CoR)
= Прибыль до налога

− Налог на прибыль (20%)
= Чистая прибыль
```

## 6. Стратегия и приоритеты

### Стратегия ВТБ (2023–2026)
- **Цифровой банк:** экосистема ВТБ Онлайн, суперапп, развитие финтеха
- **Розничный рост:** наращивание ипотеки (в т.ч. льготной — господдержка, семейная), привлечение массового сегмента
- **Импортозамещение:** финансирование промышленности, поддержка оборонно-промышленного комплекса
- **Работа с дружественными странами:** Азия, Ближний Восток, Латинская Америка — замена закрытого западного направления
- **Кросс-продажи:** страхование, НПФ, инвестиции через банковский канал

### Цифровые продукты
- **ВТБ Онлайн** — мобильный банк (30+ млн пользователей)
- **ВТБ Мои Инвестиции** — брокерское приложение (топ-3 по числу активных клиентов на Мосбирже)
- **Экосистема для бизнеса** — ВТБ Бизнес, интеграция с 1С, ЭДО

## 7. Ипотечный бизнес ВТБ

ВТБ — второй после Сбера ипотечный банк России.

**Основные программы:**
- Льготная ипотека (6,5% / 8% — по мере изменения госпрограмм)
- Семейная ипотека (6% для семей с детьми)
- IT-ипотека (5% для сотрудников IT-компаний)
- Ипотека по рыночным ставкам
- Военная ипотека

**Риски:**
- Льготные программы создают процентный риск (банк выдаёт под 6%, фондируется под 15%+) — государство субсидирует разницу
- При завершении льготных программ — резкое падение спроса
- Досрочное погашение при снижении ставок

## 8. Санкции и их влияние

**Ограничения с 2022 года:**
- Отключение от SWIFT (трансграничные платежи — через альтернативные каналы)
- Заморозка валютных активов в западных юрисдикциях
- Запрет на привлечение капитала на западных рынках (евробонды, синдикации)
- Делистинг с Лондонской биржи (LSE)
- Ограничения для иностранных контрагентов на работу с ВТБ

**Адаптация:**
- Переориентация на расчёты в рублях, юанях, дирхамах
- Использование российской СПФС (аналог SWIFT)
- Работа через банки-посредники в незападных юрисдикциях
- Рост доли рублёвого фондирования

### Нормативы ЦБ РФ (Базель III адаптированный)

|----------|------------|----------------|
| Н1.0 (достаточность общего капитала) | 8% | ~12–14% |
| Н1.1 (базовый капитал) | 4,5% | ~8–10% |
| Н1.2 (основной капитал) | 6% | ~9–11% |
| Н2 (мгновенная ликвидность) | ≥15% | выполняется |

### Особенности капитала ВТБ
- **Субординированные облигации** — инструмент пополнения капитала 2-го уровня
- **Привилегированные акции** — часть принадлежит Росимуществу (выпуск 2014–2015 в рамках докапитализации через ОФЗ)
- **Государственные докапитализации** — 2009, 2014–2015 — ВТБ получал капитал от государства в кризисные периоды

|------|-----------------|-------------|
| Сбербанк | №1 | ~33% рынка, технологический лидер, экосистема |

**Конкурентные преимущества ВТБ:**
- Доступ к крупнейшим государственным контрактам и госкорпорациям
- Масштаб и широкая продуктовая линейка
- Государственная поддержка (implicit guarantee)

**Слабые стороны:**
- Меньшая эффективность по сравнению со Сбером (CIR выше)
- Исторически более высокая стоимость риска
- Политическое давление при кредитовании госпроектов

## 11. Акции ВТБ (VTBR)

- Торгуются на Московской бирже
- Высокая доля государства → ограниченный free float
- Дивидендная политика: 50% от чистой прибыли по МСФО (целевой уровень)
- P/B исторически ниже 1 (книжная стоимость > рыночная капитализация) — типично для российских госбанков в условиях санкций
- «Народное IPO» 2007 года: цена размещения 13,6 коп. за акцию — долго торговалась ниже этого уровня, что стало символом неудачного IPO для розничных инвесторов

## 12. Ключевые термины в контексте ВТБ

| Термин | Значение |
|--------|----------|
| **ГПБ, ВЭБ** | Коллеги-госбанки в разных нишах |
| **Книга приказов (order book)** | При размещении облигаций / акций ВТБ Капиталом |
| **Андеррайтер** | ВТБ Капитал берёт на себя риск размещения выпуска ценных бумаг |
| **СПФС** | Система передачи финансовых сообщений ЦБ РФ (аналог SWIFT) |
| **СФО** | Специализированное финансовое общество (SPV для секьюритизации) |
| **Субординированный долг** | Долг, погашаемый после всех остальных кредиторов; входит в Tier 2 |
| **DFAST / ВПОДК** | Внутренние процедуры оценки достаточности капитала (российский аналог стресс-тестов ФРС) |
| **ЛГО** | Ломбардный список — перечень бумаг, принимаемых ЦБ в репо |
