# Конспект: Ключевые формулы и метрики — подготовка к собеседованию в ВТБ

---

| Блок | Метрики |

| Доходность | ROE, ROA, ROIC, NIM, NII |

## 1. БАЗОВЫЕ МЕТРИКИ ПРИБЫЛИ

### EBIT
```
EBIT = Выручка − Себестоимость − Операционные расходы
     = Операционная прибыль (прибыль до процентов и налогов)
```
> Показывает зарабатывает ли бизнес **до** влияния структуры капитала и налогов.  
> Для банков: аналог — **Операционная прибыль до резервов (PPOP)**, т.к. "выручка" у банка = NII + комиссии + трейдинг.

---

### EBITDA
```
EBITDA = EBIT + Амортизация (D&A)
```
> Прокси для операционного денежного потока. **Для банков почти не применяется** — у них нет значимых капзатрат, а главный «расход» — резервы под кредитные потери (не амортизация).

---

### NII (Net Interest Income) — Чистый процентный доход
```
NII = Процентные доходы − Процентные расходы
```
> **Абсолютная** величина: сколько рублей банк заработал на разнице ставок.  
> Основа P&L банка — обычно 55–65% всех доходов.

---

### NIM (Net Interest Margin) — Чистая процентная маржа
```
NIM = NII / Средние доходные активы × 100%

Или подробнее:
NIM = (Процентные доходы − Процентные расходы) / Средние работающие активы
```
> **Относительная** эффективность процентного бизнеса.  
> NII — это рубли, NIM — это %, они связаны через размер активов.

**Что влияет на NIM:**
- Ключевая ставка ЦБ (и скорость переоценки активов vs пассивов)
- Структура портфеля: розница даёт высокую маржу, корпораты — низкую
- Стоимость фондирования (депозиты, облигации)
- Доля недоходных активов (чем больше — тем ниже NIM)

**Важно:** при росте ключевой ставки NIM краткосрочно растёт (кредиты с плавающей ставкой переоцениваются быстрее депозитов), долгосрочно — сжимается (депозиты догоняют).

---

### EPS (Earnings Per Share) — Прибыль на акцию
```
EPS = Чистая прибыль / Количество акций в обращении

Разводнённый EPS (Diluted EPS):
EPS_diluted = Чистая прибыль / (Акции + Опционы + Конвертируемые инструменты)
```
> Показывает, сколько прибыли приходится на одну акцию.  
> Разводнённый EPS всегда ≤ базовому — учитывает потенциальное размытие при исполнении опционов.

## 2. МЕТРИКИ ДОХОДНОСТИ

### ROA (Return on Assets) — Рентабельность активов
```
ROA = Чистая прибыль / Средние активы × 100%
```
> Показывает, насколько эффективно банк использует **все** свои активы.  
> Не зависит от структуры капитала → **честнее** ROE для сравнения банков.

| Уровень ROA | Интерпретация |
|---|---|
| > 2.5% | Очень высокий (розничные банки) |
| 1.5–2.5% | Хороший |
| 0.5–1.5% | Умеренный |
| < 0.5% | Слабый |

---

### ROE (Return on Equity) — Рентабельность капитала
```
ROE = Чистая прибыль / Средний собственный капитал × 100%
```
> Показывает доходность для **акционеров**.  
> Целевой ROE крупного банка: 15–25%+

**⚠️ Ловушка ROE:** Можно искусственно повысить ROE не улучшая операции, а просто **увеличив долговую нагрузку** (leverage):
```
Если банк наращивает обязательства → активы растут → активов на 1 рубль капитала больше → ROE выше
Но: Н1 падает → регуляторный риск → при стрессе банк уязвим
```
Именно поэтому ROE всегда смотрят **вместе с** Н1 и соотношением долга к капиталу.

---

### ROIC (Return on Invested Capital) — Рентабельность вложенного капитала
```
ROIC = NOPAT / Инвестированный капитал × 100%

Где:
NOPAT = EBIT × (1 − Налоговая ставка)   [чистая операционная прибыль без долговой нагрузки]

Инвестированный капитал = Собственный капитал + Долгосрочный долг − Избыточный кэш
                        = Чистые активы, финансируемые инвесторами
```

**Для банков:** Инвестированный капитал = Регуляторный капитал (Tier 1 + Tier 2).

**ROE vs ROIC:**
| | ROE | ROIC |
|---|---|---|
| Знаменатель | Только equity | Equity + долгосрочный долг |
| Влияние leverage | Сильное | Нейтрализовано |
| Когда использовать | Оценка доходности акционеров | Сравнение компаний с разной структурой капитала |

**Правило создания стоимости:**
```
ROIC > WACC → компания создаёт стоимость (value creation)
ROIC < WACC → компания уничтожает стоимость → P/B < 1
```

## 3. МЕТРИКИ ЭФФЕКТИВНОСТИ

### PM (Profit Margin) — Маржа чистой прибыли
```
PM = Чистая прибыль / Операционные доходы × 100%
```
> Какой % дохода превращается в прибыль после всех расходов.

---

### AT (Asset Turnover) — Оборачиваемость активов
```
AT = Операционные доходы (Выручка) / Средние активы
```
> Сколько рублей дохода генерирует каждый рубль активов.  
> **Для банков:** AT низкий (~0.05–0.10) — это нормально, банки работают с большим балансом.  
> Важнее не AT сам по себе, а **NIM** как его процентная составляющая.

---

### FL (Financial Leverage) — Финансовый рычаг
```
FL = Средние активы / Средний собственный капитал
   = Мультипликатор капитала (Equity Multiplier)
```
> Показывает, во сколько раз активы превышают капитал. Сколько рублей активов на 1 рубль своих денег.

**Типичные значения:**
- Промышленная компания: FL = 2–3×
- Банк: FL = 8–15× (и это нормально, но регулируется через Н1)

**⚠️ Двойной эффект рычага:**
```
↑ FL → ↑ ROE (больше активов на тот же капитал)
↑ FL → ↓ Н1  (капитала меньше относительно RWA)
↑ FL → ↑ Риск (при убытках капитал «проедается» быстро)
```

---

### CIR (Cost-to-Income Ratio) — Коэффициент эффективности
```
CIR = Операционные расходы (OpEx) / Операционные доходы × 100%
```
> Сколько рублей тратится на каждый рубль дохода. **Чем ниже — тем лучше.**

| CIR | Оценка |
|---|---|
| < 40% | Отличная эффективность |
| 40–50% | Хорошая |
| 50–60% | Средняя |
| > 60% | Проблема с эффективностью |

> Сбер в хорошие годы достигал ~30–35%. ВТБ обычно 40–50%.

---

## 4. РАЗЛОЖЕНИЕ ДЮПОНА (DuPont Decomposition)

### 3-факторная модель (классика)
```
ROE = PM × AT × FL

ROE = (Чистая прибыль / Доходы) × (Доходы / Активы) × (Активы / Капитал)
       ───────────────────────────   ─────────────────   ─────────────────
            Маржа прибыли             Оборачиваемость      Финансовый рычаг
               (PM)                     активов (AT)           (FL)
```

**Что говорит каждый множитель:**
- **PM ↑** → банк лучше контролирует расходы или имеет более высокую маржу
- **AT ↑** → активы работают интенсивнее (больше дохода с рубля активов)
- **FL ↑** → больше заёмных денег в структуре → выше доходность на капитал, но выше риск

---

### 5-факторная модель (расширенная)
```
ROE = Tax Burden × Interest Burden × Operating Margin × AT × FL

ROE =  (ЧП/EBT)  ×    (EBT/EBIT)    ×  (EBIT/Доходы)  × (Доходы/Активы) × (Активы/Капитал)
      ──────────    ────────────────    ───────────────    ────────────────   ───────────────
      Налоговое       Процентное          Операционная       Оборачиваемость   Финансовый
       бремя           бремя                маржа              активов           рычаг
```

**Зачем 5 факторов:** позволяет **точно** диагностировать, за счёт чего изменился ROE:
- ROE упал → смотрим: вырос налог? выросли % по долгу? упала операц. маржа? упали доходы? уменьшился рычаг?

---

### DuPont для банка (специфика)
```
ROE = ROA × FL

ROA = NIM − CoR − (CIR × NIM) + Non-interest income/Assets − Налог/Активы

Раскрыто:
ROA = [NII − Резервы − OpEx + Комиссии + Трейдинг] / Активы
```

**Пример диагностики ВТБ:**
```
ROE снизился? Смотрим:
1. NIM упал (ставки? структура портфеля?)       → проблема в доходах
2. CoR вырос (ухудшилось качество портфеля?)    → проблема в рисках
3. CIR вырос (OpEx растёт быстрее доходов?)     → проблема в эффективности
4. FL снизился (нарастили капитал без роста активов?) → структурное изменение
```

## 5. ИЗ ЧЕГО СКЛАДЫВАЮТСЯ АКТИВЫ (структура RWA)

### Бухгалтерские активы
```
Активы
├── Доходные (~75–85%)
│   ├── Кредиты клиентам (корпораты + розница + МСБ)  ~60–70%
│   ├── Ценные бумаги (торговый портфель + HTM + AFS) ~15–20%
│   └── МБК выданные, депозиты в ЦБ                  ~5–10%
│
└── Недоходные (~15–25%)
    ├── ФОР (обязательные резервы в ЦБ)
    ├── Корреспондентские счета (ностро)
    ├── Основные средства и НМА
    └── Прочие (дебиторка, авансы, производные)
```

### Взвешенные по риску активы (RWA) — для расчёта Н1
```
RWA = Σ (Балансовый актив × Коэффициент риска)

Коэффициенты риска:
ОФЗ, кредиты ЦБ           → 0%   (безрисковые)
Ипотека (LTV < 80%)        → 35–50%
Кредиты корпоратам (рейт.) → 75–100%
Потребкредиты              → 100–150%
Субординированный долг     → 100%
Акции                      → 150–250%
```
> Банк может иметь большой баланс, но относительно небольшой RWA (если портфель «безопасный»). Поэтому Н1 = Капитал/RWA, а не Капитал/Активы.

## 6. МУЛЬТИПЛИКАТОРЫ ОЦЕНКИ БАНКА

### P/E (Price-to-Earnings)
```
P/E = Рыночная капитализация / Чистая прибыль
    = Цена акции / EPS
```
> Сколько лет прибыли стоит банк по рыночной цене.  
> Для банков: средний P/E ~5–12×.

---

### P/B (Price-to-Book)
```
P/B = Рыночная капитализация / Балансовый собственный капитал
    = Цена акции / Балансовая стоимость акции (BVPS)
```
> **Основной мультипликатор для банков** (в отличие от нефинансовых компаний).

| P/B | Интерпретация |
|---|---|
| > 1.5× | Рынок верит в высокий ROE, сильный бренд, рост |
| ~1× | Справедливая оценка |
| < 1× | Рынок считает, что банк **уничтожает** стоимость (ROE < COE) |

**Связь P/B с ROE:**
```
P/B ≈ (ROE − g) / (COE − g)

Где:
COE = стоимость капитала (cost of equity), обычно ~12–18% для российских банков
g   = темп роста прибыли

→ Если ROE > COE → P/B > 1 (банк создаёт стоимость)
→ Если ROE < COE → P/B < 1 (банк разрушает стоимость)
```

## 7. КРЕДИТНЫЙ РИСК — КЛЮЧЕВЫЕ ФОРМУЛЫ

### CoR (Cost of Risk) — Стоимость риска
```
CoR = Отчисления в резервы за период / Средний кредитный портфель × 100%
```
> Сколько % кредитного портфеля «теряется» в резервы.  
> Норма: 0.5–1.5%. Кризис: 2–5%+.

---

### NPL Ratio — Доля плохих кредитов
```
NPL ratio = Кредиты с просрочкой 90+ дней / Кредитный портфель брутто × 100%
```
> < 5% — приемлемо. > 10% — серьёзная проблема.

---

### Coverage Ratio — Покрытие резервами
```
Coverage = Накопленные резервы / NPL × 100%
```
> Покрытие > 100% — значит банк зарезервировал больше, чем просрочено → консервативен.  
> Покрытие < 100% — резервов не хватает → риск дорезервирования в будущем.

## 8. СВЯЗИ МЕЖДУ ФОРМУЛАМИ — МАСТЕР-СХЕМА

```
                          АКТИВЫ
                    ┌─────────────────────┐
                    │ Доходные активы     │──→ Процентные доходы ──→ NII ──→ NIM
                    │ Недоходные активы   │
                    └─────────────────────┘
                              │
                         Активы/Капитал = FL (финансовый рычаг)
                              │
┌─────────────────────────────────────────────────────────────────────┐
│  P&L: NII + Комиссии + Трейдинг = Операционные доходы              │
│        − OpEx                  = PPOP (прибыль до резервов)         │
│        − Резервы (CoR × портфель)                                   │
│        − Налог                                                       │
│        = Чистая прибыль ────────────────────────────────────────┐   │
└─────────────────────────────────────────────────────────────────│───┘
                                                                  │
          ┌───────────────────────────────────────────────────────┘
          ▼
    ROA = ЧП / Активы ◄──────────────────── «честная» доходность без рычага
    ROE = ЧП / Капитал = ROA × FL ◄──────── раздувается при росте долга
    ROIC = NOPAT / Инв. Капитал ◄─────────── нейтрализует эффект рычага
    EPS  = ЧП / Акции ◄─────────────────── доходность на 1 акцию

          ▼
    P/E  = Капитализация / ЧП ◄──────────── сколько стоит прибыль
    P/B  = Капитализация / BV ◄──────────── зависит от ROE vs COE


DuPont:  ROE  =   PM    ×   AT    ×   FL
                  ЧП/Д   Д/Активы  Акт/Кап
                  ↑          ↑         ↑
              Маржа     Оборачив.  Рычаг (риск!)
```

## 9. КРИТИЧЕСКИЕ КОММЕНТАРИИ — ЧТО ВАЖНО ГОВОРИТЬ НА ИНТЕРВЬЮ

**ROE можно «надуть» без улучшения бизнеса:**
- Взять больше долга → вырастет FL → вырастет ROE
- Сделать buyback (выкуп акций) → уменьшится капитал → вырастет ROE
- Поэтому смотри ROE **вместе с** Н1, D/E и динамикой капитала

**NIM — не только про ставки:**
- NIM зависит от **mix-эффекта**: если доля дорогой розницы растёт → NIM выше
- Банк может держать NIM стабильным даже при падении ставок, если переходит в более маржинальные сегменты

**ROIC честнее ROE:**
- Если банк рефинансирует долг дёшево → ROE выглядит отлично → но ROIC может быть низким
- ROIC > WACC — это реальный критерий создания стоимости

**P/B < 1 — сигнал тревоги:**
- Значит рынок считает: ROE < стоимости капитала (COE)
- Либо рынок не верит в устойчивость прибыли

**CIR vs рост бизнеса:**
- Банк может иметь высокий CIR временно, инвестируя в IT/отделения
- Важно: растут ли доходы быстрее расходов (операционный рычаг)?

**CoR — опережающий индикатор:**
- Рост CoR → банк видит ухудшение портфеля раньше, чем оно попадёт в NPL
- Агрессивное резервирование сейчас → меньше сюрпризов потом

**EPS легко манипулировать:**
- Buyback → меньше акций → EPS выше (без роста прибыли!)
- Всегда смотри на **рост абсолютной чистой прибыли**, а не только EPS

## 10. ВЫРУЧКА, ДОХОД, ПРИБЫЛЬ — ИЕРАРХИЯ ПОНЯТИЙ

### Определения и разграничение

| Термин | Определение | Пример |
|---|---|---|
| **Выручка (Revenue / Sales)** | Деньги, полученные от продажи товаров/услуг **до** любых вычетов | Магазин продал товаров на 1 000 000 ₽ |
| **Доход (Income)** | Широкое понятие: любое поступление экономических выгод (включает выручку + прочие доходы) | Выручка 1 млн + продали станок за 200 тыс → доход 1,2 млн |
| **Валовая прибыль (Gross Profit)** | Выручка − Себестоимость продаж (COGS) | Выручка 1 млн − COGS 600 тыс = 400 тыс |
| **Операционная прибыль (EBIT)** | Валовая прибыль − Операционные расходы (SG&A, R&D) | 400 тыс − 250 тыс OpEx = 150 тыс |
| **EBITDA** | EBIT + Амортизация (D&A) | 150 тыс + 50 тыс D&A = 200 тыс |
| **Прибыль до налогов (EBT)** | EBIT − Процентные расходы | 150 тыс − 30 тыс % = 120 тыс |
| **Чистая прибыль (Net Income)** | EBT × (1 − Налог) | 120 тыс × 0.8 = 96 тыс |

### Полная лестница P&L (Income Statement)
```
Выручка (Revenue)
    − Себестоимость (COGS)
    ─────────────────────────────
    = Валовая прибыль (Gross Profit)          → Gross Margin = Gross Profit / Revenue
    − Операционные расходы (SG&A, R&D, D&A)
    ─────────────────────────────
    = Операционная прибыль (EBIT / Operating Income)
    + Амортизация (если нужно EBITDA)
    ─────────────────────────────
    = EBITDA  [прибыль до %, налогов, амортизации]
    − Процентные расходы (Interest Expense)
    + Процентные доходы (Interest Income)
    ─────────────────────────────
    = Прибыль до налогов (EBT / Pre-tax Income)
    − Налог на прибыль
    ─────────────────────────────
    = Чистая прибыль (Net Income)
    − Дивиденды на привилегированные акции
    ─────────────────────────────
    = Прибыль, доступная акционерам → EPS = Net Income / Акции
```

### Маржинальность — ключевые формулы
```
Gross Margin      = Валовая прибыль / Выручка × 100%
Operating Margin  = EBIT / Выручка × 100%
EBITDA Margin     = EBITDA / Выручка × 100%
Net Profit Margin = Чистая прибыль / Выручка × 100%
```

> **Правило:** каждая последующая маржа ≤ предыдущей.  
> Исключение: если у компании большой процентный **доход** (депозиты, займы выданные) → EBT > EBIT → Net Margin может быть выше Operating Margin.

### NOPAT (Net Operating Profit After Tax)
```
NOPAT = EBIT × (1 − Налоговая ставка)
```
> «Операционная прибыль, как если бы у компании не было долга».  
> Используется в ROIC, EVA. Нейтрализует влияние структуры капитала.

## 11. ДЕНЕЖНЫЕ ПОТОКИ (Cash Flow Statement)

### Три раздела отчёта о движении денежных средств
```
CF от операционной деятельности (OCF / CFO)
    + CF от инвестиционной деятельности (ICF / CFI)
    + CF от финансовой деятельности (CFF)
    ─────────────────────────────────
    = Изменение денег на счёте (ΔCash)
```

### OCF (Operating Cash Flow) — операционный денежный поток
```
Косвенный метод (наиболее распространён в отчётности):
OCF = Чистая прибыль
    + Амортизация (D&A)            [неденежный расход — «возвращаем» обратно]
    + Изменение резервов/оценочных обязательств
    ± Изменение оборотного капитала (ΔWC):
        − Рост дебиторки           [деньги «заморожены» в долгах покупателей]
        − Рост запасов             [деньги «заморожены» в складе]
        + Рост кредиторки          [поставщики ждут — это «бесплатный» кредит]

Прямой метод:
OCF = Поступления от покупателей − Выплаты поставщикам − Зарплаты − Налоги
```

### ICF (Investing Cash Flow) — инвестиционный денежный поток
```
ICF = −Capex (покупка ОС и НМА)
    − Инвестиции в ценные бумаги / M&A
    + Продажа активов / дивестиции
    + Поступления от продажи ценных бумаг
```
> Обычно **отрицательный** у растущих компаний — они вкладывают деньги.

### CFF (Financing Cash Flow) — финансовый денежный поток
```
CFF = + Привлечение нового долга
    − Погашение долга
    + Эмиссия акций
    − Выкуп акций (buyback)
    − Выплата дивидендов
```

### FCF (Free Cash Flow) — свободный денежный поток
```
FCF = OCF − Capex
    = Чистая прибыль + D&A − ΔWC − Capex

Смысл: сколько денег остаётся ПОСЛЕ поддержания и развития бизнеса,
        до выплат кредиторам и акционерам.
```

### FCFF (Free Cash Flow to Firm) — для всех инвесторов (кредиторы + акционеры)
```
FCFF = EBIT × (1 − T) + D&A − ΔWC − Capex
     = NOPAT + D&A − ΔWC − Capex

Дисконтируется по WACC → используется в DCF для оценки всей компании (EV)
```

### FCFE (Free Cash Flow to Equity) — только для акционеров
```
FCFE = Чистая прибыль + D&A − ΔWC − Capex + ΔDolg (новый долг − погашение)
     = FCFF − Проценты × (1 − T) + ΔDolg

Дисконтируется по COE (стоимость капитала) → оценка стоимости акций
```

### Связи OCF ↔ Прибыль ↔ FCF
```
Чистая прибыль → OCF: добавляем неденежные расходы (D&A, резервы) и движение WC
OCF → FCF:      вычитаем капзатраты (Capex)
FCF → FCFE:     учитываем долговые транзакции
```

**Что говорит соотношение OCF и Net Income:**

| Ситуация | Вывод |
|---|---|
| OCF > Net Income | Норма: неденежные расходы велики (D&A) или WC уменьшился |
| OCF ≈ Net Income | Типично для зрелых компаний без значимой амортизации |
| OCF < Net Income | Тревога: прибыль «бумажная» — деньги в дебиторке или запасах. Возможна манипуляция |
| OCF < 0, EBITDA > 0 | WC «съедает» все доходы (быстрый рост без управления ликвидностью) |

## 12. НЕСТАНДАРТНЫЕ СЛУЧАИ — ЧТО БЫВАЕТ В РЕАЛЬНОСТИ

### Бывает ли выручка без прибыли?
**Да, и это очень распространено.**
```
Выручка есть → расходы ≥ выручки → прибыль отрицательная (убыток)
```

Примеры:
- **Amazon (2000–2012):** выручка росла двузначными темпами, операционная прибыль была ≈0 или отрицательная — всё реинвестировалось в инфраструктуру
- **Стартапы на ранней стадии:** burn rate > выручка (CAC ещё не окупился)
- **Авиакомпании в кризис:** расходы на топливо/аренду фиксированы, выручка падает → убыток
- **Банки при росте CoR:** NII есть, но резервы под убытки перекрывают доход

**Важно:** Выручка без прибыли — это не катастрофа, если бизнес-модель ведёт к окупаемости. Инвесторы смотрят на **unit economics** (прибыльность одного клиента/продукта) и **путь к прибыли**.

---

### Бывает ли доход без выручки?
**Да** — в строгом смысле «дохода» по МСФО.
```
Доход (по МСФО) = Выручка + Прочие доходы
```
Прочие доходы (без выручки от основной деятельности):
- Продажа основного средства → прибыль от продажи
- Курсовые разницы (положительные) → доход, но не выручка
- Получение субсидий → доход
- Доход от реструктуризации долга (списание части долга)

Пример: завод не продал ни одного изделия, но продал здание — доход есть, выручки нет.

---

### Бывает ли выручка без денежного потока?
**Да** — при продаже в кредит (отсрочка платежа):
```
Товар отгружен → выручка признана (по методу начисления)
Деньги не получены → OCF = 0 (или отрицательный из-за роста дебиторки)
```
Именно поэтому в P&L прибыль есть, а в Cash Flow — нет. **«Profit is an opinion, cash is a fact»** (Дракенмиллер).

Крайний случай: компания с растущей выручкой, но нарастающей дебиторкой → признак:
1. Агрессивного признания выручки
2. Проблем с качеством клиентов (не платят)
3. «Stuffing the channel» — продажи дистрибьюторам, которые потом вернут товар

---

### Бывает ли отрицательная выручка?
**Редко, но бывает:**
- **Трейдинговые компании / банки:** убыток по торговому портфелю отражается как отрицательный трейдинговый доход
- **Возвраты товаров > продаж** (технически: выручка отрицательная, но это аномалия учёта)
- **Производные инструменты:** переоценка деривативов может дать отрицательный доход от торговых операций

---

### EBITDA ≠ Денежный поток: главное заблуждение
```
EBITDA = EBIT + D&A               — это НЕ денежный поток

OCF ≈ EBITDA − ΔWC − Налоги − Проценты (упрощённо)

Разница = ΔWC + Проценты уплаченные + Налоги уплаченные
```

**Почему EBITDA обманывает:**
| Ситуация | EBITDA | FCF | Что происходит |
|---|---|---|---|
| Быстрый рост выручки | Высокий | Отрицательный | Деньги ушли в оборотный капитал (дебиторка, запасы) |
| Капиталоёмкий бизнес | Высокий | Низкий/отриц. | Capex поглощает весь операционный поток |
| Долговая нагрузка | Высокий | Низкий | % платежи не в EBITDA, но реально снижают FCF |
| Аренда (до IFRS 16) | Завышен | Реальный | Аренда «исчезала» из EBITDA как операционные расходы |

---

### Отрицательный FCF — всегда плохо?
**Нет:**
- Молодая растущая компания: FCF < 0, потому что Capex > OCF → инвестирует в будущее
- Ретейл при открытии магазинов: высокий Capex на store build-out
- Банки: понятие FCF почти не применимо — замените на FCFE или дивидендный потенциал

**Плохо**, если FCF < 0 у зрелой компании без роста — деньги «утекают» без отдачи.

---

### Чистая прибыль vs Нераспределённая прибыль vs Retained Earnings
```
Чистая прибыль (Net Income)
    − Дивиденды выплаченные
    ───────────────────────
    = Изменение нераспределённой прибыли (ΔRetained Earnings)
```
> Нераспределённая прибыль в балансе растёт на (ЧП − Дивиденды).  
> Если компания платит дивиденды больше прибыли → нераспределённая прибыль в балансе уменьшается.

---

### Для банков: аналоги выручки и OCF
| Обычная компания | Банк |
|---|---|
| Выручка (Revenue) | Операционные доходы: NII + Комиссии + Трейдинг |
| Gross Profit | NII (NII — это и "выручка" и "валовая прибыль" одновременно) |
| EBIT | PPOP (Pre-Provision Operating Profit) = Доходы − OpEx |
| Net Income | Чистая прибыль после резервов и налогов |
| FCF | Не используется. Смотрят на дивидендный потенциал = ЧП × (1 − Payout) + RWA capacity |
| OCF | Приблизительно: ЧП + Резервы − Рост кредитного портфеля |

## 13. ШПАРГАЛКА В ОДНУ СТРАНИЦУ

```
ДОХОДНОСТЬ
ROA   = ЧП / Активы                          [честная, без рычага]
ROE   = ЧП / Капитал  = ROA × FL             [для акционеров, рычагозависима]
ROIC  = NOPAT / Инв.Кап                       [нейтральна к структуре капитала]
NIM   = NII / Доходные активы                 [процентная маржа банка]

ПРИБЫЛЬ
NII   = Проц.доходы − Проц.расходы            [абсолютный % доход]
EBIT  = Выручка − OpEx (до % и налога)        [операц. прибыль]
EPS   = ЧП / Акции                            [прибыль на акцию]

ЭФФЕКТИВНОСТЬ
PM    = ЧП / Доходы                            [маржа прибыли]
AT    = Доходы / Активы                        [оборачиваемость]
CIR   = OpEx / Опер.доходы                     [↓ лучше, норма <50%]

РЫЧАГ
FL    = Активы / Капитал                       [мультипликатор капитала]

ДЮПОН
ROE   = PM × AT × FL                 (3 фактора)
ROE   = ТaxB × IntB × OpM × AT × FL  (5 факторов)

РИСК
CoR   = Резервы / Портфель                     [стоимость риска, норма <1.5%]
NPL   = Просрочка / Портфель                   [доля плохих кредитов, норма <5%]
Cov   = Резервы / NPL                          [покрытие, норма >100%]

ОЦЕНКА
P/E   = Капитализация / ЧП
P/B   = Капитализация / Балансовый капитал     [основной мульт. для банков]
```
